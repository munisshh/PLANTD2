🌱 Plant Disease Detection Using CNN

Plant Disease Detection is a deep learning-based application designed to identify whether a plant leaf is healthy or affected by disease. The project utilizes a Convolutional Neural Network (CNN), a powerful image classification technique widely used in computer vision tasks. By analyzing leaf images, the model can automatically detect disease symptoms and provide quick predictions, helping farmers and gardeners take timely action.

The system is built using Python, TensorFlow/Keras, NumPy, and Gradio. Leaf images are preprocessed through resizing and normalization before being passed to the trained CNN model. The model extracts important visual features such as color patterns, spots, and texture variations to classify the leaf as healthy or diseased. The Gradio-based web interface allows users to upload an image and instantly receive a prediction with confidence scores.

This project demonstrates the practical application of artificial intelligence in agriculture by reducing manual inspection efforts and enabling early disease detection. Early identification of plant diseases can improve crop health, increase productivity, and minimize economic losses. The project also showcases skills in deep learning, image processing, model deployment, and user interface development.

Technologies Used: Python, TensorFlow, Keras, CNN, NumPy, Gradio, Jupyter Notebook, Machine Learning, Computer Vision, and Image Classification.
